# Social Pivot - AI-Powered Social Media SaaS

A complete production-grade social media platform that leverages AI for content generation and scheduling.

## 🚀 Features

- **Multi-Tenant SaaS**: Companies and Users with isolated data.
- **AI Post Generation**: Uses OpenAI API to create engaging posts.
- **Visual Scheduler**: Drag-and-drop or table view of scheduled content.
- **Secure**: JWT Authentication and Role-based access control.

## 🏁 Quick Start

## 🔐 Security

- All API endpoints are protected with JWT.
- Webhooks are validated using a unique secret per company.
- Rate limiting is enabled on the API.
